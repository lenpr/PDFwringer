# Privacy Policy

PDFwringer processes all files locally on your device.

- **No data collection**: The app does not collect, transmit, or store any personal data or usage analytics.
- **No network requests**: The app makes no outbound network connections of any kind.
- **No third-party SDKs**: The app contains no third-party frameworks, analytics libraries, or telemetry code.
- **Local file access only**: Files are accessed only when explicitly selected by you through the system file picker or drag-and-drop. The app operates within the macOS sandbox with user-selected file access only.
- **Temporary files**: The app creates temporary files during processing (e.g., compression, color adjustment). These are stored in your system's temporary directory and cleaned up automatically.
- **Logs**: Diagnostic logs are stored locally at `~/Library/Logs/PDFwringer/` and are never transmitted. You can view or delete them at any time via Help > Show Logs.
